# Constrast Checker
Check the constrast of your background and foreground colors

![Screenshot]()

## Setup
Clone the repo:
```sh
git clone https://github.com/martinval11/contrast-checker
```

Install the dependencies using `bun`:
```
cd contrast-checker
bun install
```

Run the project:
```
bun dev
```

# Tech Stack
- Astro
- Preact
- TypeScript
- Bun
